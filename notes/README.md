# Historical notes

This directory contains archival planning notes from earlier BasicRL and cave-AI
experiments. The files here are **not** current implementation guidance,
runnable entrypoints, or production documentation.

Use these notes only as background context when writing curated docs, ADRs, or
tracked follow-up issues. Prefer adding new design material under `docs/`, in an
ADR, or in the issue tracker instead of adding new scratch files here.

## Retained files

| File | Classification | Retention reason | Deletion condition |
| --- | --- | --- | --- |
| `basicrl_project.txt` | Historical note | Captures an older BasicRL roadmap and terminology snapshot. | Delete after useful roadmap items are represented in current issues or docs. |
| `to implement.txt` | Historical note | Preserves an early cave-species AI sketch that may inform future R&D triage. | Delete after relevant AI ideas are promoted to ADRs, current-status docs, or issues. |
